# Danish Ali — Portfolio Website

A polished, responsive developer portfolio built as a single-page static website. It showcases Danish Ali's DevOps and cloud engineering skills, project experience, academic background, and contact information.

## Overview

This project is a personal portfolio site written in plain HTML, CSS, and JavaScript. It includes:

- A hero section with professional branding and contact links
- Technical skills and tools overview
- Featured DevOps, cloud, and infrastructure projects
- Education section
- Contact panel with email, phone, GitHub, and LinkedIn links

## Features

- Responsive layout for desktop and mobile
- Animated section reveal and stats counters
- Inline CSS for fast loading with modern design
- Supported external links to GitHub and LinkedIn profiles
- Built for easy local preview and hosting as a static page

## Usage

1. Clone or download the repository.
2. Open `index.html` in a web browser.

Optional:

- Host the site on GitHub Pages or any static file host.

## Notes

- The site includes inline CSS and JavaScript in `index.html`, with no external build step required.
- Project links in the portfolio section point to GitHub repositories for real infrastructure and DevOps work.
